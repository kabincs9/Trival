A dynamic and engaging **Trivia Quiz** web application built with vanilla HTML, CSS, and JavaScript.

## ✨ Features

- ✅ Multiple-choice trivia questions
- ✅ Real-time score tracking
- ✅ Progress indicator (Question X of Y)
- ✅ Immediate feedback (correct/incorrect answers)
- ✅ Clean and modern user interface
- ✅ Responsive design (works on mobile & desktop)
- ✅ Restart quiz functionality
- ✅ Smooth animations and transitions

## 🎮 How to Play

1. Clone the repository
2. Open `index.html` in your browser
3. Answer each question by selecting one of the options
4. See your score update in real-time
5. View final results at the end of the quiz
